# AgriPulse Precision Agriculture
AI-powered soil analysis for the HackArena 2026 challenge.

## Features
- **Logic Engine:** Implements Critical Crop Penalties and Universal Thresholds.
- **JSON Compliance:** Outputs data in the mandatory clinical schema.
- **UI:** Streamlit dashboard with color-coded suitability metrics.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run app: `streamlit run app.py`
